# Requirements

## Architecture
- x86_64 (packaged and tested)
- Docker required

## Hardware
- 4GB RAM minimum, 8GB recommended
- 1GB storage
- Standard CPU (no special hardware)

## Software
- Docker 20.10+
- Linux/macOS/Windows

## Dependencies
All dependencies included in Docker image:
- Python 3.9
- numpy, scipy, sympy, networkx, matplotlib, pandas

## Performance
- Basic examples: <30 seconds
- Full experiments: ~5 minutes
- Memory usage: <1GB
